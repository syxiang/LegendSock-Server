Quantumult 实际是 Cross 的浴火重生版，以前的 rixCloud Documents 将其描述为新生代应用的说法是错误的。

Quantumult 提供对 SSR 协议的支持，并提供和 Surge 相当的稳定性和可靠性，同时其售价远低于 Surge，符合其本身价值。

Quantumult 是 rixCloud 推荐在 iOS 设备上使用的客户端。
